• Z-Anatomy ↗

    The Z-Anatomy app is a free and open source 3D human anatomy atlas that can be used on desktop computers (PC/WebGL) and Android devices.
    It is a useful tool for students and professionals in the medical field who want to study human anatomy in a more interactive and visual way.
    It is the largest and most ambitious project I have ever done, and the one that has had the most impact worldwide.
    I developed it independently without funding.

  • EZ-Crosshair

    This application allows the user to create super customizable crosshairs to play shooter-type video games.
    It offers very advanced features such as changing shape, color, size, rotation, as well as custom animations and many other things.
    It is still in the development phase and will soon be released on Steam.

  • Sun Shader ↗

    Shader that simulates a star using millions of particles.
    It is programmed with ShaderGraph (Unity).
    Educational purpose project.

  • Planet Shader ↗

    The background shader for this web page.
    Nope, it's not a video, it's real time generated shader.
    It has been programmed in GLSL.

  • Ecosystem Simulation ↗

    Basic simulator of a simple ecosystem in which there are organisms and predators that compete to survive.
    Educational purpose project.

  • Fourier Series Visualization

    This project allows the user to visualize the sum of several cosine waves.
    I programmed this tool to gain deeper knowledge about how the Fourier transform works and its applications.
    Educational purpose project.

  • Py-Crosshair

    This project is a prototype of an application to create and use crosshairs in video games.
    It was later replaced by the EZ-Crosshair project due to the inflexibility of python for creating desktop applications.

  • Hand To Binary ↗

    • Receives the video through the camera using the cv2 library.

    • Recognizes hands through the mediapipe library.

    • Calculate which fingers are raised and which are not by calculating the angle between landmarks on the hands.

    • Map raised fingers to binary numbers.

  • Face Comparison

    It is a library that provides the user with an easy-to-use interface to:

    • Capture and save faces from images, videos or through the camera.
    • Train the neural network to catalog the different faces and associate them with a name.
    • Recognize faces present in photos, videos or through the camera.
